Sphinx-doc
==========

`Sphinx-doc` 不是 `Sphinx`_ (全文搜索引擎), 这里是指一种文档编写工具. 基于 reStructureText 文本格式. 使用 python 编写

.. _Sphinx: http://sphinxsearch.com/
